# 🚀 Arcade Nexus

**Arcade Nexus** is a modern, responsive gaming hub designed to bring your favorite web-based games together in one place. It provides a sleek, unified interface that makes switching between different titles fast, smooth, and visually engaging.

---

## ✨ Features

* **Centralized Hub:** Access your entire library of games from a single, organized dashboard.
* **Seamless Game Switching:** Use the persistent sidebar to jump between games instantly without reloading the entire website.
* **Immersive Experience:** A dedicated, distraction-free viewing area designed to keep the focus on the gameplay.
* **Clean Dark Aesthetic:** A "Night Mode" inspired design that reduces eye strain and fits the gaming vibe.
* **Fully Responsive:** Optimized for various screen sizes, ensuring the arcade looks great on tablets and desktops alike.

---

## 🛠️ How it Works

The website functions as a smart "wrapper" for web games. By utilizing a configuration file, the site knows exactly which games are available and where they are located. When a user selects a game from the sidebar, the application dynamically loads that game into the main viewing window. 

This architecture allows the hub to stay lightweight and fast, as it only loads the specific game you want to play at that moment.

---

## 🎮 Vision

Arcade Nexus was created to solve the problem of cluttered bookmarks and disorganized game files. By providing a structured environment, it turns a collection of individual files into a professional-grade gaming platform. It is built to be "evergreen," meaning new games can be added to the collection at any time to keep the arcade fresh and exciting.
